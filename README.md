# Batlokoa Innovative Projects Website

Modern, 3D-enhanced website for Batlokoa Innovative Projects - a 100% Black-Women-Owned, Level 1 BBB-EE engineering supply company.

## Tech Stack

- **React 18** + Vite
- **Three.js** + React Three Fiber (3D)
- **Framer Motion** + GSAP (Animations)
- **Tailwind CSS** (Styling)
- **React Hook Form** + Zod (Forms)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/
│   ├── 3d/          # Three.js components
│   ├── layout/      # Header, Footer, Layout
│   ├── ui/          # Buttons, Cards, Badges
│   └── sections/    # Page sections
├── pages/           # Route pages
├── hooks/           # Custom React hooks
├── utils/           # Utilities
├── data/            # Static data
├── styles/          # Global styles
└── assets/          # Images, 3D models, icons
```

## Environment Variables

Create a `.env` file:

```
VITE_FORMSPREE_ID=your_formspree_id
VITE_GA_TRACKING_ID=your_ga4_id
```

## Deployment

Deploy to Vercel:

```bash
npm run build
# Deploy dist/ folder
```
